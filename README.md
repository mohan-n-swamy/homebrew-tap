# homebrew-tap

Personal [Homebrew](https://brew.sh) tap.

```bash
brew tap mohan-n-swamy/tap
```

## Formulae

| Formula | Description |
|---|---|
| [`coding-control-tower`](https://github.com/mohan-n-swamy/coding-control-tower) | Local, project-first dashboard for coding sessions and pull requests |
| [`tmux-llm-dashboard`](https://github.com/mohan-n-swamy/tmux-llm-dashboard) | Nine-row LLM quota + usage dashboard for a tmux pane |

```bash
brew install mohan-n-swamy/tap/coding-control-tower
brew install mohan-n-swamy/tap/tmux-llm-dashboard
```
